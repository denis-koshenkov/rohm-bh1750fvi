# Rohm BH1750FVI Driver
Non-blocking driver for Rohm BH1750FVI light intensity sensor in C language.

# Running Tests
Prerequisites:
- CMake
- C compiler (e.g. gcc)
- C++ compiler (e.g. g++)

Steps:
1. Clone this repository
2. Fetch the CppUTest submodule:
```
git submodule init
git submodule update
```
3. Build and run the tests:
```
./run_tests.sh
```
